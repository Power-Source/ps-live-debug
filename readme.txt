== Description ==

PSOURCE Live Debug ist ein praktisches Plugin für ClassicPress und WordPress, das das Debugging vereinfacht. Es aktiviert den Debug-Modus und integriert einen eigenen Admin-Bildschirm, auf dem das aktuelle debug.log direkt im Backend eingesehen werden kann. So können Fehler und Warnungen schnell erkannt und behoben werden, ohne auf den Server zugreifen zu müssen.

**Features:**
- Aktiviert automatisch den Debug-Modus (`WP_DEBUG`, `WP_DEBUG_LOG`, `WP_DEBUG_DISPLAY`)
- Fügt einen Menüpunkt im Adminbereich hinzu, um das debug.log einzusehen
- Ermöglicht das einfache Löschen des debug.log aus dem Backend
- Übersichtliche Darstellung der letzten Log-Einträge
- Kompatibel mit ClassicPress und WordPress
- Keine Konfiguration notwendig – sofort einsatzbereit

Ideal für Entwickler und Administratoren, die Fehler schnell und unkompliziert analysieren möchten.
